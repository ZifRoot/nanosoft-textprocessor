# Nanosoft TextProcessor

**Nanosoft TextProcessor** — это приложение для обработки текстовых файлов, позволяющее удалять короткие слова и знаки препинания. Программа поддерживает как одиночные файлы, так и пакетную обработку, эффективно работает с большими объёмами данных благодаря потоковому чтению/записи и не блокирует интерфейс пользователя.

## Основные возможности

- **Обработка одного или нескольких файлов** — выбор файлов или целой папки с текстовыми форматами (`.txt`, `.csv`, `.log`, `.md` и др.).
- **Гибкая настройка правил**:
  - минимальная длина сохраняемого слова;
  - удаление знаков препинания с выбором режима (простое удаление, замена на пробел, «умная» замена);
  - настройка набора символов пунктуации, учёт цифр, дефисов, апострофов;
  - нормализация пробелов и сохранение переносов строк.
- **Пакетная обработка с параллельным выполнением** — несколько файлов обрабатываются одновременно (степень параллелизма настраивается).
- **Потоковая обработка больших файлов** — без загрузки целиком в память.
- **Прозрачный прогресс** — отображение процента выполнения, скорости и оставшегося времени.
- **Расширяемая архитектура** — паттерн «Стратегия» позволяет добавлять новые алгоритмы обработки без изменения ядра.
- **Современный интерфейс** на WPF (MVVM), не зависающий при длительных операциях.
- **Модульные тесты** покрывают ключевую логику библиотеки.

## Технологии

- .NET 8.0
- Библиотека классов (TextProcessor.Lib)
- WPF (TextProcessor.UI) с паттерном MVVM
- xUnit / NUnit для тестирования

## Быстрый старт

1. Склонируйте репозиторий.
2. Откройте решение `TextProcessor.sln` в Visual Studio 2022 или другой среде с поддержкой .NET 8.
3. Соберите решение (восстановление пакетов NuGet выполнится автоматически).
4. Запустите проект `TextProcessor.UI`.

Подробное техническое задание и описание архитектуры см. в [SPECIFICATION.md](SPECIFICATION.md).
